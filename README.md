# 🎬 IMDb-Inspired Movie & TV Show Database  

## 📌 Overview  
This is a **full-stack web application** inspired by **IMDb**, designed for movie and TV show enthusiasts. Built with **Django (backend)** and **HTML, CSS, JavaScript (frontend)**, this platform offers a **seamless, feature-rich experience** where users can **search, rate, and manage** their favorite content while staying updated with real-time IMDb ratings.  

The platform also provides **one-click access** to where movies and TV shows can be streamed and allows users to watch official **YouTube trailers** directly within the app.  

## ✨ Key Features  
✔ **🔍 Search for Movies & TV Shows** – Find content by title, genre, category, or release year  
✔ **⭐ Rate & Review** – Leave ratings and reviews (IMDb ratings sync automatically)  
✔ **🎥 Add Custom Listings** – Users can contribute new movies and TV shows  
✔ **📝 Comment on Listings** – Engage with the community by discussing movies/shows  
✔ **🎞️ Watchlist Management** – Add or remove items from your personal watchlist  
✔ **👤 User Authentication** – Sign up, log in, or browse without an account  
✔ **🔄 Real-Time IMDb Rating Sync** – Automatically updates IMDb ratings  
✔ **▶ Watch Trailers** – View official YouTube trailers directly within the platform  
✔ **📍 Where to Watch** – One-click access to streaming platforms for each movie/show  
✔ **📱 Fully Mobile Responsive** – Optimized for all devices  

## 🛠️ Tech Stack  
### **Frontend**  
- **HTML5, CSS3, JavaScript** – Responsive and dynamic UI  
- **JavaScript** – Handles interactivity and dynamic content updates  

### **Backend**  
- **Django (Python)** – Manages authentication, database interactions, and API integrations  
- **SQLite** – Lightweight database for efficient data storage
